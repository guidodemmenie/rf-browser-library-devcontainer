# Prerequisites

- Docker
- VS Code

## Steps
1. Install Podman Desktop [https://podman-desktop.io]
2. Instal VS Code [https://code.visualstudio.com/download]


#### VS Code
1. F1 -> settings
1.  dev.containers.dockerPath
1. zoek:  Dev containers  settings
1.    setting  Docker path
1.      -> replace docker by podman

clone repo

restart after closing all
  open folder   rf-browser-library-devcontainer
F1  rebuild and reopen container
